Navigation drawer

Navigation drawers lets users navigate the entire content of a product or a section. These can be used for a single level or a multi-level navigation.

Single level

Use a single level navigation when you have a simple or flat navigation, without a hierarchy. Choosing a navigation item will select the item and take the user to the corresponding location

Single level with groups

Use a single level with group / section headers when you need to group navigation items into categories. This variant has the same behavior as the single-level side navigation, except that it has headers that aren’t interactive. If some items don’t fall into one of the existing categories, place them at the top of the drawer

Multi-level

Use a multi-level navigation when you have multiple layers of hierarchical navigation. Clicking on a header opens or collapses the sub-level navigation items, and in some cases also sends the user to the top-level location
