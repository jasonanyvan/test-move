Tooltips

A tooltip is a floating, non-actionable label used to explain or provide context to a UI element

Small

The default size for tooltips

Medium

An alternative size for tooltips that need a bit more impact

Detailed

As an edge case, tooltips can contain up to 90 characters if needed

Nudge

A variation on the tooltip is the nudge. Use these where you need to draw attention to a part in the UI, usually a new feature, and want to do it contextually. These have arrows to point to specific parts of the UI, where are standard tooltips get directionality from hovers or clicks, and are dismissable by a user with a text button

The default positioning for tooltips should be top but sometimes it is necessary to reposition them for improved readability. Tooltips should appear on hover and on focus
