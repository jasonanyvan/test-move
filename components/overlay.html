Overlays

An overlay is the translucent background that is used by other components or to improve readability

An overlay is a fixed position element with entrance and exit animation

Open overlay

For improved readability of content placed on or in compontents, the image can be darkened with an overlay

Use increments perfectly divisible by 3 for the overlay's alpha value - .75 should be the default
