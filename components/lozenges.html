Lozenges

A lozenge is a visual indicator used to highlight an item's status for quick recognition

Default

Lozenges are always bold and the use color to indicate meanings is very important as users learn and recognize these colours and meanings across products, platforms and executions

EARLY

Truncation

By default, the maximum width of a badge is 200px regardless of the size. When the text in the badge exceeds the max width, it will be truncated with an ellipsis.

Avoid truncation wherever possible by using shorter text in badges. The truncated text is not focusable or accessible.
